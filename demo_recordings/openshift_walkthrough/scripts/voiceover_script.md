# OpenShift Platform Demo Walkthrough

## Demo Overview
A comprehensive walkthrough of Red Hat OpenShift Container Platform running on AWS, demonstrating the key features and capabilities for managing containerized workloads.

**Total Duration:** ~3-4 minutes
**Cluster:** Red Hat OpenShift Service on AWS (ROSA)
**Version:** OpenShift 4.19.21

---

## Step 1: Introduction & Cluster Overview (~30s)

**Narration:**
"Welcome to Red Hat OpenShift Container Platform. In this walkthrough, we'll explore the key features of OpenShift running on AWS. This is a production-ready Kubernetes platform that provides enterprise-grade container orchestration with built-in developer and operations tools."

**Visual:** Cluster Overview dashboard showing status, details, and activity
**Actions:** Show dashboard, highlight cluster details panel

---

## Step 2: Cluster Details & Status (~25s)

**Narration:**
"Here on the Overview page, you can see your cluster health at a glance. Notice the cluster is running on AWS infrastructure with OpenShift version 4.19.21 and a Premium SLA tier. The status panel shows cluster health, Insights recommendations, operator status, and real-time activity logs."

**Visual:** Overview page with details and status panels
**Actions:** Point out cluster API, version, infrastructure provider, status indicators

---

## Step 3: Projects & Namespaces (~30s)

**Narration:**
"OpenShift organizes workloads into Projects, which extend Kubernetes namespaces with additional security and resource management. Here we can see multiple projects including AI workloads like kagent, agentic-workflows, and the NVIDIA GPU operator for machine learning acceleration. Each project shows its resource consumption including memory and CPU usage."

**Visual:** Projects list with resource metrics
**Actions:** Scroll through projects, highlight AI-related ones

---

## Step 4: Workloads & Pods (~35s)

**Narration:**
"Let's look at the running workloads. In the Pods view, you can monitor all containers running in your cluster. Notice the AI agent pods including code-agent, docs-agent, and kagenti services. Each pod displays its status, resource consumption, restart count, and creation time. This gives operators full visibility into application health."

**Visual:** Pods listing with status and metrics
**Actions:** Show pods list, highlight status column, resource usage

---

## Step 5: Operators & Marketplace (~30s)

**Narration:**
"OpenShift Operators extend the platform with enterprise capabilities. Here we see Red Hat OpenShift AI installed, which enables data science and machine learning workloads. Also installed is OpenShift Service Mesh, providing advanced networking features like traffic management and observability powered by Istio."

**Visual:** Installed Operators page showing OpenShift AI and Service Mesh
**Actions:** Show installed operators, highlight their provided APIs

---

## Step 6: Summary (~20s)

**Narration:**
"That concludes our walkthrough of Red Hat OpenShift. From centralized cluster management to AI-powered workloads and enterprise operators, OpenShift provides everything you need to build, deploy, and scale modern applications. Thank you for watching."

**Visual:** Return to Overview dashboard
**Actions:** Final view of dashboard

---
